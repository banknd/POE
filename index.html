Path of Exile Autoflask & AutoQuit

Only works on the stand-alone version of the game found at pathofexile.com. How to use this:

If you want to run the script from the .ahk file:

Download and install the Unicode 32-bit version of AutoHotkey. Download the script from github. Download Currports. (This is what we use to log out) Put Autopot.ahk, AutoHotkeyMemoryLib.ahk, and cports.exe in the same folder. Run Autopot.ahk (Make sure AutoHotkey.exe is being run as admin) If you simply want to run an exe:

Download the exe from github. Download Currports. (This is what we use to log out) Put AutoPot.exe (rename it if you want) and cports.exe in the same folder. Run AutoPot.exe (as admin) Short video of it in action
